# Contributing to Aethereal Nexus

Thank you for your interest in contributing to Aethereal Nexus. This project is free, open-source, and community-driven.

- Fork the repository and create a feature branch for your work.
- Keep changes small and focused; open a pull request describing the intent and impact.
- Write clear commit messages and include relevant screenshots or recordings for UI changes.
- Follow existing CSS/JS structure and use `style.css` imports for new styles.
- For internationalization, add localized content under `content/{lang}/`.

If you find an issue, please open an issue with a short reproducible description.

Thank you — Bruno & the Aethereal Nexus community
